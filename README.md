# 🚀 HackHustle: The Ultimate Interview Prep Platform

**HackHustle** is a comprehensive, full-stack platform designed to bridge the gap between learning and landing a dream job. It features a dual-user ecosystem for both students and instructors, powered by AI and real-time collaboration.

---

## 🌟 Key Features

### 🎓 For Students (Candidates)
* **⚔️ Quiz Battles:** Real-time competitive coding and aptitude challenges.
* **🤖 AI Mock Interviews:** Simulated interview sessions with automated feedback (Powered by FastAPI).
* **📚 MCQ Practice:** Comprehensive question bank for DS, OOPS, and DBMS.
* **❓ Doubt Resolution:** Integrated system to ask questions and get expert guidance.

### 👨‍🏫 For Teachers (Instructors)
* **📊 Teacher Dashboard:** Dedicated account to view resolved doubts, rating, number of students mentored etc.
* **💡 Doubt Management:** Real-time interface to resolve student queries and provide mentorship.


---

## 🛠️ Tech Stack

### Frontend
- **React.js**: Dynamic UI with specialized dashboards for Students and Teachers.
- **Tailwind CSS**: For a clean, professional look.

### Backend (Microservices)
- **Spring Boot (Java)**: Handles core logic, user roles (Student/Teacher), and MCQ management.
- **FastAPI (Python)**: Powers the AI Interview engine and high-speed data tasks.

### Database & Tools
- **MySQL**: Structured storage for user profiles, mcq questions and doubt history.
- **Axios**: Seamless API communication across services.
